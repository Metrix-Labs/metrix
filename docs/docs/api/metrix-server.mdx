# Metrix Server

:::info

Current state: **Stable**

:::

The Metrix server module permits to generate a Metrix http server.

## Module: Metrix server

### `createServer(metrix)`

- `metrix`: [<Type>Metrix</Type>](Metrix)
- Returns: <Type>StrapiServer</Type>

```javascript
const server = createServer(metrix);

server.listRoutes();
```

### `StrapiServer.app`

- [<Type>KoaJS</Type>](https://devdocs.io/koa/index)

Metrix projects are using KoaJS to run the NodeJS server.

### `StrapiServer.router`

- [<Type>@koa/router</Type>](https://github.com/ZijianHe/koa-router#router-)

Metrix projects are using a dependency of KoaJS called @koa/router.

### `StrapiServer.httpServer`

- [<Type>http.Server</Type>](https://nodejs.org/docs/latest-v18.x/api/http.html)

The Metrix's HTTP server.

### `StrapiServer.api(name)`

- `name`: <Type>String</Type>
- Returns: [<Type>StrapiAPIs</Type>](#strapiapis)

Getter for apis available in Metrix

### `StrapiServer.use(...args)`

- [<Type>KoaApp.use</Type>](https://devdocs.io/koa/index#appusefunction)

Shortcut for Koa `app.use(...args)` method.
